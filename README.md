# DATA_MINING_PROJECT
Data mining project for CSE3054!

#### Pre-requisites
- [Python](https://www.python.org/)
- [Open CV](https://opencv.org/)


#### Installation

1. Clone the repo
```sh
git clone https://github.com/Vish-19/DATA_MINING_PROJECT.git
```
3. Move into folder
```sh
cd DATA_MINING_PROJECT
```
4. Run the commands
```sh
python <file_name>
```
